# FoodHub Database System

This project is a C# Windows Forms and Microsoft SQL Server based database system developed for the FoodHub Company scenario in Unit 04 – Database Design & Development.

## Project Overview
The system was designed and implemented to manage FoodHub's delivery operations at the initial stage of business expansion in Sri Lanka. It supports customer registration, telephone order processing, food item and ingredient management, rider management, motor bike allocation, dispatch recording, and related database operations.

## Technologies Used
- C#
- Windows Forms
- Microsoft Visual Studio 2022
- Microsoft SQL Server
- SQL Server Management Studio

## Main Features
- Customer registration, update, delete and search
- Food item and ingredient management
- Telephone order entry with multiple order items
- Dispatch time recording
- Rider management
- Rider dependent management
- Motor bike management with multiple colours
- Bike assignment tracking with meter readings
- SQL validations, constraints, triggers and relational integrity

## Database Structure
Main tables included:
- Customer
- Orders
- FoodItem
- Ingredient
- FoodItemIngredient
- OrderItem
- Rider
- Dependent
- MotorBike
- BikeColour
- BikeAssignment

## How to Run the Project
1. Open SQL Server Management Studio.
2. Run the scripts in the `DatabaseScripts` folder in this order:
   - `01_Create_Database.sql`
   - `02_Sample_Data.sql`
   - `03_Views_Triggers_Security.sql`
3. Open the solution file in Visual Studio 2022.
4. Check the connection string in `App.config`.
5. Build and run the project.

## Documentation
The project documents are available in the `Documentation` folder.
